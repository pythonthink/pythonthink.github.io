python

    mode='r',  # 操作模式， read、write、append
    buffering=None,
    encoding=None,  # 文件编码，常用的是 gbk 与 utf-8
    errors=None,
    newline=None,
    closefd=True

各个参数的细节如下：

file ​

操作文件的位置(文件保存在电脑的路径)

  • 绝对路径: 从盘符的根目录开始指定的文件路径 D:\qd\qd1
  • 相对路径: 相当于我当前编辑py文件位于哪一个文件夹而言
  • 如果文件无法打开，则会引发 OSError

mode ​

文件操作的模式 指定你需要用什么方式操作这个文件

  • 其他常见的值是w用于写入（如果文件已经存在，则覆盖该文件）
  • a用于附加（在某些 Unix 系统上，这意味着无论当前的搜索位置如何，所有写操作都会附加到文件末尾）
  • 默认为r，表示使用文本的方式打开文件来读取
  • b二进制数据的写入和读取 图片/视频/音频 wb rb
  • x用于独占创建

encoding ​

编码

  • window utf-8 gbk
  • mac utf-8
  • linux utf-8

在文本模式下，如果未指定编码，则使用的编码与平台相关：调用locale.getpreferredencoding(False)以获取当前语言环境编码。（为了读取和写入原始字节，使用二进制模式并且不用指定编码）可用的模式有：

字符含义
'r'用于读取（默认）
'w'用于写入，首先覆盖文件
'x'用于独占创建，如果文件已经存在则失败
'a'用于写入，追加到文件末尾（如果存在）
'b'二进制模式
't'文本模式（默认）
'+'打开磁盘文件进行更新（读取和写入）

对于二进制读写访问，模式w+b打开并将文件删减为0字节。r+b打开文件而不删减

Python区分二进制和文本I/O。可以以二进制模式打开的文件（ mode 参数中包括 b）将内容作为字节对象返回，而不进行任何解码

arr = ['1', '2', 3, 4, 5, '6', '7']
# 第一步 将列表里面的数据类型 全部都强转成数值类型 在进行数值的加1
# 方法一
# map 它是会默认把序列的每一个元素进行便利的
# map(函数的名字,序列)
def func(x):
    # print(x)
    s = int(x) + 1
    print(s)
    print(type(s))
    return s # 实现函数内外的数据共享
result = map(func,arr)

# join 是将序列合并成新的序列(是字符串的方法)
# 将数据转化为str类型
# 因为join是字符串的方法所以 我现在就需要将列表里面的每一个元素再一次转回str
# result2 = ','.join(result1)
# 列表推导式
content = ','.join([str(i) for i in result])
print(content)
print(type(content))

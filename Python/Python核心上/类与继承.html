字数: 0 字 时长: 0 分钟

编写类时，并非总是要从空白开始。如果你要编写的类是另一个现成类的特殊版本，可使用继承。一个类继承另一个类时，它将自动获得另一个类的所有属性和方法；原有的类称为父类，而新类称为子类 。子类继承了其父类的所有属性和方法，同时还可以定义自己的属性和方法

继承的概念：子类默认继承父类的所有属性和方法

DANGER

在Python中，所有类默认继承object类，object类是顶级类或基类；其他子类叫做派生类

子类继承自父类，可以直接享受父类中已经封装好的方法，不需要再次开发

子类中应该根据职责，封装子类特有的属性和方法

        # 实例属性
        self.secret = '[古法煎饼果子配方]'

    # 定义方法函数

子类重写父类实例属性 ​

python
"""狮子"""

class Lion:
    def __init__(self, name, weight, high):
        self.name = name
        self.weight = weight
        self.high = high
    # 定义方法函数
    def eat(self):
        return f'{self.name}正在吃东西'
    
class Cat(Lion):
    # 用于在不改变父类的属性结构上 在子类里面去添加属于自己的属性
    def __init__(self, name, weight, high, color):
        # 通过super绑定我们的属性,绑定到当前对象里面去
        # 不改变父类的情况下 去扩展子类自己的属性
        # super()  会在底层默认传递self 用于一个单继承 跟类对象不同 类对象是多继承 所以super()不用去传递self 它会默认传递self
        super().__init__(name, weight, high)
        self.color = color  # 新增的属性也是需要绑定到子类对象里面去的.它是子类所独有的属性,父类里面是没有的
        # name, high, weight是从父类里面继承过来 color是绑定我当前子类对象里面，所以color是子类特有的

# 对子类实例化
cat = Cat('辛巴','50cm','60kg','棕色')  # 参数的数量 父类的参数数量 + 子类新增的   4个

# print(cat.eat())
# print(cat.color)
>>> print(cat.name)
cat.name = '辛巴1号'  # 可以对外部进行一个重新赋值
>>> print(cat.name)

辛巴
辛巴1号

# 对父类本身进行实例化
lion = Lion('辛巴','50cm','60kg') # 此时对父类本身传递只要三个参数 没有子类
print(lion.name)
print(lion.color)  # 这里会报错 说明这个属性已经是子类独有的了

子类重写父类的方法函数 ​

"""狮子"""


class Lion:
    def __init__(self, name, weight, high):
        self.name = name
        self.weight = weight
        self.high = high
    # 定义方法函数
    def eat(self):
        return f'{self.name}正在吃东西'
class Cat(Lion):
    # 用于在不改变父类的属性结构上 在子类里面去添加属于自己的属性
    def __init__(self, name, weight, high, color):
        #  这种方法后面再仔细讲解 有个印象
        # 通过super绑定我们的属性,绑定到当前对象里面去
        # 不改变父类的情况下 去扩展子类自己的属性
        # super()  会在底层默认传递self
        super().__init__(name, weight, high)
        self.color = color  # 新增的属性也是需要绑定到子类对象里面去的.它是子类所独有的属性,父类里面是没有的
        # name, high, weight是从父类里面继承过来 color是绑定我当前子类对象里面
    # 同名的 实例属性是不会往后面查找的 但是方法函数是可以的

"""
炼体 PracticeBody  class
    属性：     init  属性绑定到初始化函数里面去的
        名字
        血量(blood)
        体力(power)

    行为：       方法函数 def
        挑水（carry_water）
            挑水消耗2点体力
        砍柴（chop_wood）
            砍柴消耗3点体力

练气 PracticeMagic           可以让练气期去继承练体期 练体期里面的 name  blood power 都会被继承过来
                              super在子类(练气期)去额外扩展一个灵力属性就OK了
    属性： 
        name
        血量
        体力
      + 灵力（magical_power）
    行为：
        挑水
            挑水消耗0.2点灵力
        砍柴
            砍柴消耗0.3点灵力
      + 御风(fly)
            御风消耗2点灵力
      + 喷火(spurt_fire)
            喷火消耗2点灵力

练神 PracticeDivine         炼神期去继承练气期    super在子类里面增加神力属性就OK了


    属性：
        血量
        体力
        灵力
      + 神力(super_power)
      可以通super()进行添加的
    行为：
      - 挑水
      - 砍柴
        御风  fly
            御风消耗0.2点神力
        喷火
            喷火消耗0.2点神力
      + 御剑(flying_sword)
            御剑飞行消耗 2 点神力
"""

class PracticeBody:
    """练体期"""

    def __init__(self, name, blood, power):
        self.name = name
        self.blood = blood
        self.power = power

    # 挑水
    def carry_water(self):
        self.power -= 2  # 每调用一次消耗2点体力

    # 砍树
    def chop_wood(self):
        self.power -= 3  # 每调用一次消耗3点体力
        return '消耗了3点体力值'


class PracticeMagic(PracticeBody):  # 炼气期继承练体期
    """炼气期"""

    def __init__(self, name, blood, power, magical_power):  # 新增灵力属性
        super().__init__(name, blood, power)  # 继承父类的3个属性
        self.magical_power = magical_power  # 将新增的灵力属性绑定到当前的子类对象里面去 子类独有

    # 挑水
    def carry_water(self):
        self.magical_power -= 0.2  # 每调用一次消耗0.2点灵力

    # 砍树
    def chop_wood(self):
        self.magical_power -= 0.3  # 每调用一次消耗0.3点灵力

    # 御风
    def fly(self):
        self.magical_power -= 2  # 每调用一次消耗2点灵力

    # 喷火
    def spurt_fire(self):
        self.magical_power -= 2  # 每调用一次消耗2点灵力


class PracticeDivine(PracticeMagic):  # 炼神期去继承炼气期
    def __init__(self, name, blood, power, magical_power, super_power):  # 新增神力属性
        super().__init__(name, blood, power, magical_power)  # 继承父类的属性
        self.super_power = super_power  # 将新增的神力属性绑定到当前的子类对象里面去

    # 挑水的函数
    def carry_water(self):
        return '炼神期不需要这种低阶技能'

    # 砍树
    def chop_wood(self):
        return '炼神期不需要这种低阶技能'

    # 御风
    def fly(self):
        self.super_power -= 0.2  # 每调用一次消耗0.2点神力

    # 喷火
    def spurt_fire(self):
        self.super_power -= 0.2  # 每调用一次消耗0.2点神力

    # 御剑风行
    def flying_sword(self):
        self.super_power -= 2  # # 每调用一次消耗2点神力

子类init方法 ​

创建子类的实例时，Python首先需要完成的任务是给父类的所有属性赋值。为此，子类的方法__init__()需要父类施以援手

例如，下面来模拟坦克。 坦克英雄是一种特殊的对象，因此我们可以在前面创建的Hreo类的基础上创建新类Tank，这样我们就只需为坦克特有的属性和行为编写代码

    def __init__(self, name, defence, blood):
        # self.name = name

        # 在完成仪式的时候，爸爸有的属性儿子需要传递给爸爸  

        self.defence = defence
        self.blood = blood


tank = TanK('坦克', 100, 1000)

class Cat(Animal):

    def info(self):
        print(f'种类:猫\t名字:{self.name}\t颜色:{self.color}\t体重:{self.weight}')

"""
请用面向对象的继承的方式实现以下类的封装:

    动物类（Animal）：  class
        属性：name, high, weight  初始化函数
        行为：吃

    老虎类（Tiger）： 取继承  Animal 只需要取封装老虎的狩猎技能
        属性：name, high, weight
        行为：吃、老虎的狩猎技能

    狮子类（Lion）：    取继承  Animal     只需要取封装狮子的狩猎技能
        属性：name, high, weight
        行为：吃、狮子的狩猎技能

    狮虎兽（Liger）：
        属性：name, high, weight 多继承 子类里面去调用其他父类的方法
        行为：吃、既有老虎的狩猎技能、也有狮子的狩猎技能    类对象去调用多个父类的方法和属性
"""


# 定义动物类
class Animal:
    def __init__(self, name, high, weight):
        self.name = name
        self.high = high
        self.weight = weight

    # 定义吃的函数
    def eat(self):
        return f'{self.name}正在吃东西'


# 定义老虎类
class Tiger(Animal):  # 老虎 继承动物 此时动物的方法和属性都会被继承
    # 封装老虎的狩猎技能
    def hunt(self):
        return f'{self.name}发动了老虎的狩猎技能'


# 定义狮子类
class Lion(Animal):  # 狮子 继承动物 此时动物的方法和属性都会被继承
    # 封装狮子的狩猎技能
    def hunt(self):
        return f'{self.name}发动了狮子的狩猎技能'


# 狮虎兽
class Liger(Lion, Tiger):
    # 封装老虎的狩猎技能
    def tiger_hunt(self):
        # 类对象调用是不会默认传递self
        Tiger.__init__(self,self.name,self.high,self.weight)
        return Tiger.hunt(self)
        # super().__init__(self.name, self.high, self.weight)
        # return super().hunt()

    # 封装狮子的狩猎技能
    def lion_hunt(self):
        Lion.__init__(self,self.name,self.high,self.weight)
        return Lion.hunt(self)
    #     super().__init__(self.name, self.high, self.weight)
    #     return super().hunt()

python
"""
作业1
在第九次作业Hero(英雄类)的基础上，新增了法师英雄，多了下面内容(考察super用法与多继承)

法师（Mage）
    增加属性：魔法值（magical 默认0，最大100）   super去添加魔法值

    - 攻击：  需要自己去重写 方法函数
        调用一次 怒气 `+2`
        调用一次 魔法值 `+5`
    - 放大招    if 判断 是否达到释放大招的条件
      魔法值满时自动放大招

    - 第二形态 自己封装的一个函数
      当怒气值满时 自动切换第二形态（魔法值最大值修改为50）

"""


class Hero:
    def __init__(self, name, weapon, equipment, power, blood, anger):
        self.name = name
        self.weapon = weapon
        self.equipment = equipment
        self.power = power
        self.blood = blood
        self.anger = anger
        self.max_anger = 100  # 如果我们的anger达到100 就可以释放大招

    # 用于释放大招的方法函数
    def nirvana(self):
        self.anger = 0  # 释放完进行清零的操作
        print(f'{self.name}释放了大招')

    # 定义用于攻击的方法函数
    def attack(self):
        print(f'{self.name}发动了攻击')
        self.anger += 2  # 每攻击一次 怒气值加2
        if self.anger >= self.max_anger:  # 条件满足,可以释放大招
            self.nirvana()  # 触发释放大招的函数


# 魔法师继承英雄类 魔法师类已经继承了英雄类的实例属性和方法函数
class Mage(Hero):
    def __init__(self, name, weapon, equipment, power, blood, anger, magical):  # 新增魔法值的属性
        super().__init__(name, weapon, equipment, power, blood, anger)  # 继承父类的实例属性
        self.magical = magical  # 专属于子类的属性 绑定当前的子类对象里面去
        self.max_magical = 100  # 定义触发大招的魔法值的上限

    # 封装用于切换第二形态的函数
    def second(self):
        self.anger = 0
        print(f'{self.name}切换了第二形态')
        self.max_magical = 50
    # 重写攻击的函数
    def attack(self):
        print(f'{self.name}发动了攻击')
        self.anger += 2  # 每攻击一次 怒气值加2
        self.magical += 5  # 每攻击一次 魔法值加5
        if self.magical >= self.max_magical:  # 达到了触发大招的条件
            self.nirvana()
        if self.anger >= self.max_anger:  # 可以切换第二形态
            self.second()
# 实例化
mage = Mage('小乔', '扇子', ['衣服', '鞋子'], 10, 100, 96,35)

mage.attack()
print('怒气值:',mage.anger)
print('魔法值:',mage.magical)

mage.attack()
print('怒气值:',mage.anger)
print('魔法值:',mage.magical)

mage.attack()
print('怒气值:',mage.anger)
print('魔法值:',mage.magical)
